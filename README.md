# 6_sem_coursework

## Запуск приложения

1. Клонировать репозиторий:

```bash
git clone https://github.com/Olljux/6_sem_coursework
```

2. Перейти в папку проекта:
```bash
cd 6_sem_coursework
```

4. Собрать Docker-образ:
```bash
docker compose build
```

5. Запустить контейнер:
```bash
docker compose up -d
```
